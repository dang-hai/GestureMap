# Setup 

1. Install python requirements

```bash
> pip install -r requirements.txt
```

2. Install frontend application dependencies:

```bash
> cd frontend && npm i
```

3. Run backend

```
> python backend/app.py
```

4. Run frontend

```
> cd frontend && npm start
```

The application should now run on localhost:3000
